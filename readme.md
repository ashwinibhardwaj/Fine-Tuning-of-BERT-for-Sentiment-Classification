# 🚀 BERT Sentiment Analysis (Hugging Face + PyTorch)

This project demonstrates fine-tuning a **BERT model** for binary sentiment classification using the Hugging Face ecosystem.

---

## 🔗 Hugging Face Model

👉 https://huggingface.co/ashwini10521/finetuned_bert  

---

## 📌 Project Overview

- Fine-tuned a pretrained Transformer (**BERT**) on IMDb dataset  
- Built a complete NLP pipeline: preprocessing → training → evaluation → inference  
- Evaluated model performance using standard classification metrics  
- Visualized results using a confusion matrix  
- Deployed the trained model on Hugging Face Hub  

---

## 🧠 Model Details

- **Base Model:** `bert-base-uncased`  
- **Task:** Binary Sentiment Classification  
- **Framework:** PyTorch + Transformers  
- **Dataset:** IMDb  

---

## 📊 Results
